USING BACKBONE'S PARSE() METHOD

Heres a quick note on how to use Backbone’s parse method, since the documentation seemed fuzzy…

Parse is useful when you want to change or rename model data between the server and the client side. Parse is called automatically when data is returned from the server, as happens with fetch() or save().

For example, you might want a property called ‘uniqueID’ on the server to appear as simply ‘id’ in your models:

var Users = Backbone.Collection.extend({
  parse: function(response)
    response.forEach(function(user) {
      user.id = user.uniqueID;
      delete user.uniqueID;
    });
    return response;
}
});

Response, in the case above, will be the collection returned from the server from a fetch call, in the form of an array of objects. We are just looping through the array and making changes as needed. Whatever we return from the parse method becomes the data that will actually be stored in the collection.

Following is an example where we want to move and rename some values.  For example, instead of accessing the company name via user.company.name, we want to access it with user.company_name.

var Users = Backbone.Collection.extend({

  model: User,
  parse: function(response) {
    response.forEach(function(user) {
      user.company_name = user.company.name;
      user.catchPhrase = user.company.catchPhrase;
