WHAT DOES 'VAR' ACTUALLY DO?

Primarily, var lets javaScript know that you are declaring a variable in the current scope.  If we declare a variable without var, javaScript will look up the scope chain for that variable.  If it doesn’t find it, it will simply create the variable in the global context, on the window object.

function testVar() {
  withoutVar = 'hello';
  var withVar = 'hello';
}
 
testVar();
console.log(withoutVar); // 'hello'
console.log(window.withoutVar);  // 'hello'
console.log(withVar); // ReferenceError: withVar is not defined

The var keyword (line 3) created the variable within the function’s scope, making it inaccessible outside the function.  withoutVar was actually created as a property of the window object.
