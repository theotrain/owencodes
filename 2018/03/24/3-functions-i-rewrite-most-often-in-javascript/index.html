Constrain Amount

Sometimes you want a number to stay in certain bounds. We could keep the value of ‘sprite.x’ within 0..100 like so:

function constrainToRange(value, min, max) {
return (Math.min(max, Math.max(min, value)));
}

sprite.x = constrainToRange(sprite.x + deltaX, 0, 100)

function randomInRange(min, max) {
return min + Math.floor(Math.random() * (max - min + 1));

function random(min, max) {
if (max == null) {
max = min;
min = 0;
}
return min + Math.floor(Math.random() * (max - min + 1));
};

…which allows you the option of passing only one argument to get a number between 0 and the argument.

Deep Copy

Often i need to copy something like an array with objects in it. Javascript doesn’t provide a way to do this easily, and a naive attempt is likely to create multiple pointers to the same objects, rather than proper copies.

Recursion to the rescue! This deep copy function i stole from this post by Avijit Gupta.

JavaScript

function copy(o) {
var output, v, key;
output = Array.isArray(o) ? [] : {};
for (key in o) {
v = o[key];
output[key] = (typeof v === "object") ? copy(v) : v;
}
return output;
}
